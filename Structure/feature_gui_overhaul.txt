#                    FEATURE: Programmatic GUI Overhaul                        #
################################################################################

================================================================================
DESCRIPTION
================================================================================

Conversion de tous les 10 écrans de containers du rendu par texture (blit sur
ResourceLocation) au rendu programmatique (GuiGraphics.fill() + drawString()).

Avantages:
- Plus besoin de fichiers PNG pour les backgrounds GUI
- Modification des layouts directement dans le code
- Style cohérent garanti par GuiRenderHelper partagé
- Réduction des assets (suppression textures GUI)

================================================================================
UTILITAIRES PARTAGÉS
================================================================================

GuiRenderHelper.java (client/gui/):
- renderContainerBackground(): fond gris #C6C6C6 avec bordures 3D (highlight/shadow)
  + séparateur horizontal + labels (titre container + "Inventory")
- renderSlot(): slot 18x18 avec effet inset (3 couleurs: shadow, highlight, fond)
- renderPlayerInventory(): 27 slots (3x9) + 9 slots hotbar avec espacement
  (DÉPRÉCIÉ: remplacé par PlayerInventoryWidget dans les 10 screens)
- renderProgressArrow(): flèche 22x16 honey-gold (#E8A317) avec remplissage progressif
- renderProgressBar(): barre horizontale avec fill ratio
- getFluidName(): extraction nom depuis FluidStack

FluidGaugeWidget.java (client/gui/widget/):
- Widget réutilisable pour jauges de fluide
- Frame inset, fill basé sur ratio, graduations 25/50/75%
- Tooltip avec nom fluide, montant, pourcentage
- Couleurs par type de fluide (honey, royal_jelly, nectar)

PlayerInventoryWidget.java (client/gui/widget/):
- Widget réutilisable pour rendu inventaire joueur via texture (blit)
- Texture: textures/gui/inventory.png
- Remplace le rendu programmatique de GuiRenderHelper.renderPlayerInventory()
- Intégré dans les 10 screens (8 alchemy + 2 hive)
