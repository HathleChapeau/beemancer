--------------------------------------------------------------------------------
FEATURE: Debug Bee System
--------------------------------------------------------------------------------
Description: Système d'abeille debug avec stats et contrôle via baguette
Statut: COMPLÈTE

Scripts créés:
- common/entity/bee/DebugBeeEntity.java : Entité abeille avec stats HP/Speed/Strength
- common/item/bee/BeeDebugItem.java : Item pour spawn/capture avec tooltip stats
- common/item/bee/BeeWandItem.java : Baguette pour sélection et navigation
- common/event/BeeInteractionHandler.java : Gestion shift+clic pour capture

Fonctionnalités:
- Stats: HP (vert), Speed (bleu), Strength (rouge) - défaut 1
- BeeDebugItem: clic droit = spawn, shift+clic droit sur abeille = capture
- Tooltip: "Press SHIFT for more detail" -> affiche stats colorées
- BeeWandItem: 
  - Clic droit sur abeille = sélection (son Ding)
  - Clic droit sur bloc = définir destination (son Dong)
  - Shift+clic droit air = désélectionner
  - Tooltip affiche entité sélectionnée
- L'abeille se déplace vers la destination définie

Registres:
- BeemancerItems: +bee_debug, +bee_wand
- BeemancerEntities: +debug_bee

Assets:
- models/item/bee_debug.json, bee_wand.json
- textures/item/bee_debug.png, bee_wand.png
- lang/en_us.json, fr_fr.json (traductions complètes)

Référence: Pattern Entity inspiré de Cobblemon
