DESCRIPTION
================================================================================

Système de tiers pour les machines d'alchimie et de logistique.
Chaque tier améliore les performances (capacité, vitesse, débit).

================================================================================
ARCHITECTURE
================================================================================

APPROCHE: Paramètres exposés, pas de nouvelles classes.

Les BlockEntity existants ont été modifiés pour accepter des paramètres de tier:
- Constructeur avec paramètres (capacity, transferRate, etc.)
- Factory methods statiques pour chaque tier (createTier2, createTier3...)
- Constantes statiques pour les valeurs de chaque tier

Les Block existants stockent le tier et délèguent au bon BlockEntity:
- Champ `private final int tier`
- newBlockEntity() retourne le bon tier via factory method
- getTicker() retourne le bon BlockEntityType selon le tier
