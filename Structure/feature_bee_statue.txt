################################################################################
#                         FEATURE: BEE STATUE                                  #
################################################################################

================================================================================
DESCRIPTION
================================================================================

Bloc décoratif servant de statue d'abeille avec piédestal. Affiche une espèce
d'abeille en rotation avec son nom au-dessus. Permet de parcourir les espèces
et d'obtenir des abeilles avec les gènes par défaut.

================================================================================
INTERACTIONS
================================================================================

- Shift + Clic droit: Cycle vers l'espèce suivante (affiche "X/N")
- Clic droit: Donne une MagicBee de l'espèce actuelle avec gènes par défaut

================================================================================
RENDU
================================================================================

- Piédestal: Modèle bloc standard (base + colonne + plateau)
- Abeille: Modèle vanilla BeeModel rendu par BeeStatueRenderer
  - Position: Centre du bloc, au-dessus du plateau
  - Rotation: Lente rotation continue (2°/tick)
  - Échelle: 0.7 (légèrement plus petite qu'une abeille normale)
- Nom: Text flottant au-dessus de l'abeille
  - Face toujours le joueur (billboard)
  - Fond semi-transparent

================================================================================
NOTES TECHNIQUES
================================================================================

- Le BlockEntity stocke uniquement l'ID de l'espèce (String)
- Les données sont synchronisées via NBT (getUpdateTag/handleUpdateTag)
- Le renderer utilise le modèle vanilla BeeModel avec texture custom ou fallback
- La texture cherche: textures/entity/bee/{speciesId}_bee.png, sinon vanilla
