################################################################################
#                         FEATURE: POLLEN POT                                  #
################################################################################

================================================================================
DESCRIPTION
================================================================================

Pot de stockage pour pollens. Peut contenir jusqu'à 16 pollens d'un seul type.
Interaction simple: clic droit pour ajouter, clic gauche pour retirer.

================================================================================
FONCTIONNEMENT
================================================================================

AJOUT DE POLLEN (Clic droit):
- Le joueur doit tenir un pollen dans sa main
- Si le pot est vide: accepte n'importe quel pollen
- Si le pot contient déjà du pollen: accepte uniquement le même type
- Consomme 1 pollen de la main du joueur (sauf mode créatif)
- Maximum 16 pollens par pot

RETRAIT DE POLLEN (Clic gauche):
- Retire 1 pollen du pot
- Le pollen est donné directement au joueur (inventaire ou drop)
- Si le pot est vide: comportement normal (casser le bloc)
- L'event est annulé pour éviter de casser le bloc

DESTRUCTION DU BLOC:
- Drop tout le contenu (pollens) en plus du bloc lui-même

MESSAGES:
- Pot vide: "Pot is empty"
- Contenu: "Contains X/16 [Pollen Name]"
- Pot plein: "Pot is full!"
- Mauvais type: "This pot already contains a different pollen type"

================================================================================
POLLENS SUPPORTÉS
================================================================================

Tag: beemancer:pollens

Items inclus:
- pollen (générique)
- flower_pollen
- mushroom_spore
- tree_pollen
- crystal_pollen
- pollen_of_wind
- pollen_of_thunder
- pollen_of_water
- pollen_of_fire
- pollen_of_light
- pollen_of_darkness
- void_pollen

================================================================================
TEXTURES DES POLLENS (Rendu dynamique)
================================================================================

L'intérieur du pot affiche un cube texturé dont la hauteur est proportionnelle
au nombre de pollens stockés (0-16). Utilise un BlockEntityRenderer + textures
de concrete powder vanilla. Mapping défini dans PollenTextures.java et
pollen_textures.json.

| Pollen              | Texture                              |
|---------------------|--------------------------------------|
| flower_pollen       | minecraft:block/pink_concrete_powder |
| mushroom_spore      | minecraft:block/brown_concrete_powder|
| tree_pollen         | minecraft:block/green_concrete_powder|
| crystal_pollen      | minecraft:block/light_blue_concrete_powder |
| pollen_of_wind      | minecraft:block/light_gray_concrete_powder |
| pollen_of_thunder   | minecraft:block/yellow_concrete_powder|
| pollen_of_water     | minecraft:block/blue_concrete_powder |
| pollen_of_fire      | minecraft:block/orange_concrete_powder|
| pollen_of_light     | minecraft:block/white_concrete_powder|
| pollen_of_darkness  | minecraft:block/purple_concrete_powder|
| void_pollen         | minecraft:block/black_concrete_powder|
| (défaut)            | minecraft:block/light_gray_concrete_powder |

================================================================================
ARCHITECTURE
================================================================================

FICHIERS:
- common/block/pollenpot/PollenPotBlock.java
  - Bloc avec shape custom (pot)
  - useItemOn() pour clic droit
  - onRemove() pour drop contenu
  - Support Comparator (signal 0-15)

- common/block/pollenpot/PollenPotBlockEntity.java
  - Stockage: ItemStack (type) + int (count)
  - Méthodes: addPollen(), removePollen(), canAcceptPollen()
  - NBT serialization
  - Sync client/server via getUpdateTag() + handleUpdateTag()
  - getComparatorOutput() pour signal redstone

- common/block/pollenpot/PollenPotEvents.java
  - Event handler pour clic gauche (LeftClickBlock)
  - Intercepte et annule si pollen retiré
  - Vérification MAIN_HAND pour éviter double exécution

- core/registry/BeemancerTags.java
  - Définition du tag POLLENS pour items

- client/color/PollenTextures.java
  - Mapping pollen -> texture (11 textures concrete powder)
  - Texture par défaut: light_gray_concrete_powder

- client/renderer/block/PollenPotRenderer.java
  - BlockEntityRenderer avec cube texturé
  - Hauteur dynamique proportionnelle au remplissage
  - Utilise FluidCubeRenderer + PollenTextures

REGISTRES:
- BeemancerBlocks.POLLEN_POT
- BeemancerBlockEntities.POLLEN_POT
- BeemancerItems.POLLEN_POT (BlockItem)
