################################################################################
#                     FEATURE: ALCHEMY PROCESSING SYSTEM                        #
################################################################################

================================================================================
DESCRIPTION
================================================================================

Systeme de traitement alchimique modulaire base sur des recettes JSON.
Supporte les inputs/outputs d'items et de fluides avec probabilites configurables.

================================================================================
FORMAT RECETTES JSON
================================================================================

CENTRIFUGING:
```json
{
  "type": "apica:centrifuging",
  "ingredient": { "item": "apica:common_comb" },
  "results": [
    { "item": "apica:pollen", "chance": 1.0 },
    { "item": "apica:beeswax", "chance": 0.5 }
  ],
  "fluid_result": { "fluid": "apica:honey", "amount": 250 },
  "processing_time": 60
}
```

INFUSING:
```json
{
  "type": "apica:infusing",
  "ingredient": { "tag": "minecraft:logs" },
  "fluid_ingredient": { "fluid": "apica:honey", "amount": 250 },
  "result": { "item": "apica:honeyed_wood" },
  "processing_time": 200
}
```

CRYSTALLIZING:
```json
{
  "type": "apica:crystallizing",
  "fluid_ingredient": { "fluid": "apica:honey", "amount": 500 },
  "result": { "item": "apica:honey_crystal" },
  "processing_time": 100
}
```

DISTILLING:
```json
{
  "type": "apica:distilling",
  "fluid_ingredients": [
    { "fluid": "apica:honey", "amount": 500 },
    { "fluid": "apica:royal_jelly", "amount": 250 }
  ],
  "fluid_result": { "fluid": "apica:nectar", "amount": 500 },
  "processing_time": 80
}
```

================================================================================
UTILISATION
================================================================================

AJOUTER UNE RECETTE:
1. Creer un fichier JSON dans data/apica/recipe/[type]/
2. Suivre le format correspondant au type de recette
3. /reload pour charger la nouvelle recette

CREER UN NOUVEAU TYPE DE MACHINE:
1. Creer Recipe (extends Recipe<ProcessingRecipeInput>)
2. Enregistrer RecipeType dans ApicaRecipeTypes
3. Enregistrer Serializer dans ApicaRecipeSerializers
4. Creer BlockEntity utilisant le nouveau type de recette
5. Creer Menu avec ApicaSlot
6. Creer Screen avec FluidGaugeWidget
